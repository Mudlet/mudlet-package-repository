> Mudlet Package Repository

All Packages

The following is a list of all packages available in the Mudlet Package Repository.

AchaeaChatTabs

by Erikka, Demonnic, version 1.3.3



Chat tabs for Achaea

MedUI

by Kymbahl, Kronos, version 1.7.8



User Interface for Medievia

MudletBusted

by Demonnic, version 2.0.0



The Busted testing suite, packaged for Mudlet

NumPad

by Oneymus, version 1



Cardinal movement commands assigned to number pad keys.

command-line-api

by Zooka, version 1



Search and open Mudlet API webpage from the command line

cron-daemon

by Zooka, version 1



A cron daemon/scheduler to perform tasks at fixed times, dates or intervals.

delay-scrolling

by Zooka, version 1.1



Slow down the main window buffer giving you time to read.

deleteOldProfiles

by Mudlet Default Package, version 1



Remove excess backup files.

dice

by Zooka, version 1



Simulate dice rolls with these simple functions.

echo

by Mudlet Default Package, version 1



A set of aliases to test triggers on the command line.

enable-accessibility

by Mudlet Default Package, version 1



Configuration for visually impaired users.

generic_mapper

by Mudlet Default Package, version 2.1.4



Semi-automatic mapping, designed to work with many MUDs.

gui-drop

by Mudlet Default Package, version 1



Drag and drop images onto the main window to turn into a label and container.

master-spam

by Zooka, version 1



Suppress lines of spamminess.

mpkg

by Zooka, version 2.8



A command line package manager for Mudlet.

run-lua-code

by Mudlet Default Package, version 5



Run Lua code directly from the command line.

shaxamm

by Xanthia, version 1.0



Xanthia's Audio Manager Module

simple-mapper

by Zooka, version 2.1



Create simple maps using directional commands only.

speedWalkRecorder

by demonnic (mods by Zooka), version 2.3



Records a speedwalk with the ability to play back, and in reverse.
