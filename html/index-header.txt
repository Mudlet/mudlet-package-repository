				<h2>What is this?</h2>
				<p>mpkg is a command line package manager for Mudlet which allows you to install, remove, update and search for
				packages in this repository.  Packages like mappers, GUI's, tutorials and helpful functions, all created by Mudlet
				users.  Think of it as a software manager for Mudlet.</p>
			</article>
			<article>
				<h2>Installation</h2>
				<p>To access this repository from within Mudlet copy/paste the following to your command line;<br><br>
				<strong>lua installPackage("https://mudlet.github.io/mudlet-package-repository/packages/mpkg.mpackage")</strong></p>
			</article>
			<article>
				<h2>Further help</h2>
				<p>Once installed access the help via:<br><br>
				<strong>mpkg help</strong></p>
